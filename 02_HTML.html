글씨를 굵게 만드는 태그 'b' 태그와 'strong' 태그

b 태그 는 글씨를 굵게 만들어준다. 다만 표현상으로만 굵게 해주기 때문에 strong 태그와 달리 강조 표현으로 사용하지는 않는다. Strong 태그 는 글씨를 굵게 만들어주지만, 표현에만 그치지 않고, 사용자에게 강조 표현이라는 것을 알려준다.

문단을 만들어 주는 태그 'p' 태그

p 태그는 문단을 만들어 주는 태그이다. 만약 세 개의 문단을 만들고자 한다면 'p' 태그를 3개 사용하면 된다. 사용한 갯수만큼 만들어진다. p 태그만 사용하면, 문단과 문단 사이를 자동으로 띄워주기 때문에 사용하기 아주 좋다.

제목을 만들어 주는 'h1' 태그

h1 태그는 제목 태그로, h1 태그를 활용하면, 강제로 제목에 맞는 크기가 되어서 표현을 해준다. h1 ~ h6 태그가 있으며, 크기가 커질수록 글씨 크기는 반대로 작아진다.

리스트 태그인 'ol', 'ul', 'li'

- 'ol' 태그는 순서가 있는 리스트를 말한다. 그래서 1, 2 이런식으로 순서가 진행이 된다.
- 'ul' 태그는 순서가 없는 리스트를 말한다. 그래서 특수문자 기호 형식으로 순서가 진행이 된다.
- 'li' 태그는 ol 과 ul 를 뒷받침 해줄 수 있게 항목을 만들어 준다.
- 리스트 태그는 html 의 중첩에 대해 말을 할 때, 대표적인 예시로 사용이 되는 태그이다.
